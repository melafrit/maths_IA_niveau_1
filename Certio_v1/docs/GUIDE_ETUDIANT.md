## 1. Avant l'examen

### Ce que votre prof vous a envoyé

Un email (ou message) avec :

```
📬 Votre examen est programmé

📝 Titre : Contrôle vecteurs L1
📅 Date : 25 avril 2026
⏰ Horaires : de 14:00 à 16:00
⏱️ Durée : 60 minutes
🔑 Code d'accès : ABC23K-9P
🌐 URL : https://examens.ipssi.fr/etudiant/passage.html
```

### Checklist de préparation

Avant le jour J :

- [ ] **Tester le lien** : ouvrir l'URL pour vérifier qu'elle charge bien
- [ ] **Navigateur à jour** : Chrome, Firefox, Safari ou Edge récent
- [ ] **Connexion stable** : Wi-Fi ou câble (éviter 4G qui peut couper)
- [ ] **Batterie chargée** : ou branchement secteur
- [ ] **Notifications désactivées** : Slack, email... pour pas perdre de temps
- [ ] **Fermer autres onglets** : les pop-ups peuvent déclencher une anomalie
- [ ] **Crayon + papier** : pour brouillon si besoin

### 5 min avant l'horaire

- Ouvrir le lien donné par le prof
- Préparer : nom, prénom, email

## 3. Pendant l'examen

### Interface principale

```
┌──────────────────────────────────────────────────────────┐
│  Contrôle vecteurs L1       ⏱️ 47:23 restant            │
├──────────────────────────────────────────────────────────┤
│                                                          │
│   Question 3 / 20                                       │
│                                                          │
│   Quelle est la dérivée de f(x) = x² en x = 3 ?         │
│                                                          │
│   ○ A.  6                                               │
│   ● B.  9                      ← votre réponse          │
│   ○ C.  2                                               │
│   ○ D.  x + 2                                           │
│                                                          │
│   [← Précédent]   [1][2][3][4]...[20]   [Suivant →]     │
│                                                          │
│                           [ 📤 Soumettre l'examen ]     │
└──────────────────────────────────────────────────────────┘
```

### Fonctionnalités

#### 🕐 Chronomètre
- Affiché en haut à droite
- **Auto-submit** quand il atteint 0:00
- Les 5 dernières minutes : compte à rebours **rouge clignotant**
- Les 1 dernières minutes : **alarme sonore**

#### 🧭 Navigation entre questions
- Boutons **Précédent** / **Suivant**
- Cliquer directement sur un **numéro de question** (1, 2, 3...)
- Ordre **aléatoire** (pas le même pour tout le monde)

#### 💾 Sauvegarde automatique
- Vos réponses sont **enregistrées à chaque clic**
- Pas besoin de "sauvegarder" manuellement
- Si vous perdez la connexion 30s, tout est déjà stocké

#### 🎨 Indicateurs visuels

| Couleur | Signification |
|---|---|
| 🔵 Bleu | Question **répondue** |
| ⚪ Gris | Question **non répondue** |
| 🟣 Violet | Question **actuelle** |
| 🟡 Jaune | Question **marquée pour revoir** |

Vous pouvez "marquer" une question pour y revenir plus tard (icône drapeau).

#### 📱 Mobile
L'interface s'adapte au téléphone, mais **préférez un ordinateur** pour la saisie et la lecture.

### ⚠️ À éviter pendant l'examen

Certaines actions déclenchent une **anomalie** (loggée et vue par le prof) :

| Action | Anomalie | Gravité |
|---|---|:-:|
| Copier (Ctrl+C) | `copy` | ⚠️ |
| Coller (Ctrl+V) | `paste` | ⚠️ |
| Ouvrir Devtools (F12) | `devtools_open` | 🚨 |
| Changer d'onglet | `blur` | ⚠️ |
| Clic droit | `right_click` | ℹ️ |

**Conseils** :
- Gardez cet onglet **actif** (pas d'aller-retour)
- **Pas de Ctrl+C/V** : le sujet est autonome
- **Pas de F12** ni de "Inspecter"

### En cas de problème

#### "J'ai fermé l'onglet par erreur"
- Rouvrez l'URL et ressaisissez vos infos
- **Important** : utilisez le **même email** que précédemment !
- Si votre session est encore active (< durée max), vous reprenez où vous étiez
- Vos réponses précédentes sont conservées

#### "Internet a coupé"
- Pas de panique : vos dernières réponses sont sauvegardées
- Reconnectez-vous, rafraîchissez (F5)
- Resaisissez vos infos, vous retrouvez votre passage

#### "Le chrono est presque à zéro et je n'ai pas fini"
- **Soumettez ce que vous avez** (bouton "Soumettre")
- Ne pas soumettre = expiration auto avec ce qui est sauvegardé
- Dans les deux cas : vos dernières réponses comptent

## 4. Soumettre son examen

### Quand soumettre ?

- Quand vous avez **fini** (toutes les questions répondues)
- **ou** quand vous ne savez plus répondre
- **ou** quand le chrono arrive à zéro (auto)

## 6. Questions fréquentes

### ❓ Puis-je utiliser une calculatrice ?

Dépend du prof. Par défaut, pas d'interdiction technique — à vérifier dans la consigne.

### ❓ Puis-je utiliser papier / brouillon ?

Oui. Le sujet ne l'interdit pas techniquement.

### ❓ Puis-je consulter mes cours ?

⚠️ Tout dépend du contrôle :
- **Examen fermé** : NON, même si techniquement possible
- **Quiz ouvert / formatif** : autorisé si précisé par le prof

En cas de doute → **demander au prof**.

### ❓ Combien de temps dure le lien de correction ?

Le token dans l'URL est valide **plusieurs jours** (configurable par le prof). En cas de doute, sauvegarder le PDF dès que possible.

### ❓ Puis-je repasser l'examen ?

Par défaut : **non** (max_passages = 1). Certains profs activent plusieurs tentatives — précisé dans la consigne.

### ❓ Mon score n'apparaît pas, pourquoi ?

- La correction peut être **différée** (30 min, 1h, après clôture...)
- Check votre **email** pour le lien direct
- Si > 24h sans accès : contacter le prof

### ❓ J'ai fait une faute d'orthographe sur mon nom

Pas grave — le compte rendu peut être corrigé côté prof. Lui envoyer un email.

### ❓ Est-ce que je peux revenir en arrière dans l'examen ?

**Oui** pendant l'examen (boutons Précédent / clic sur numéro).
**Non** après soumission.

### ❓ Puis-je sauvegarder un brouillon et reprendre plus tard ?

- Pendant l'examen, tout est auto-sauvegardé
- Si vous fermez et rouvrez, vous reprenez où vous étiez (tant que le chrono tourne)
- Mais vous **ne pouvez pas fermer l'examen puis le reprendre dans 3h** — le chrono tourne en continu

### ❓ Puis-je imprimer les questions avant de soumettre ?

Techniquement possible, mais fortement déconseillé :
- Utiliser **Ctrl+P** ou clic droit peut déclencher une **anomalie**
- Les profs peuvent considérer cela comme tentative de sortir du contenu

## 7. Problèmes techniques

### 🐛 Le code d'accès est "invalide"

**Vérifications** :
- Respect des **majuscules/minuscules** (case sensitive)
- Copier-coller depuis l'email (éviter fautes)
- Code tapé avec **tirets** : `ABC23K-9P` et non `ABC23K9P`
- Si période ouverture dépassée → message spécifique

Si problème persiste → contacter le prof.

### 🐛 Écran blanc / ne charge pas

1. **Rafraîchir** la page (F5)
2. Essayer en **navigation privée**
3. Tester un **autre navigateur**
4. Désactiver **extensions** (adblock, noscript...)
5. Vérifier votre **connexion** : `ping 8.8.8.8`

### 🐛 Erreur 429 "Too many requests"

Vous avez envoyé trop de requêtes trop vite.

- Attendre **1 minute**
- Arrêter les F5 intempestifs
- Rafraîchir une seule fois

### 🐛 Erreur 500 "Internal Server Error"

Problème côté serveur (pas de votre fait).

- Attendre quelques minutes
- Rafraîchir
- Si persistant : contacter l'admin IPSSI

### 🐛 Le chrono est figé

- **Ne pas paniquer** : vos réponses sont sauvegardées
- Rafraîchir (F5)
- Se reconnecter avec le même code + email
- Le chrono reprendra où il en était

### 🐛 Les formules math ne s'affichent pas

- Vérifier que **JavaScript est activé** dans le navigateur
- Rafraîchir la page
- Vider le cache navigateur (Ctrl+Shift+R)
- Tester sur un autre navigateur

### 🐛 Le site est lent

- Vérifier votre connexion
- Fermer les autres onglets lourds
- Désactiver les extensions navigateur
- Essayer depuis un autre réseau si possible

### 🐛 Je pense avoir été injustement "invalidé"

- Les anomalies sont **loggées mais pas forcément bloquantes**
- Un prof review avant validation définitive
- Si refus définitif → contestation par **email** au prof avec explications

## 🎯 Bonnes pratiques résumées

✅ **À faire** :
- Lire calmement chaque question
- Gérer votre temps (2-3 min/question)
- **Répondre** à toutes les questions (pas de "blanc" si possible)
- **Marquer pour revoir** les questions difficiles et y revenir
- **Soumettre avant le chrono** si vous avez fini

❌ **À éviter** :
- Paniquer si chrono descend
- Paniquer si internet coupe 10s
- Changer d'onglet / ouvrir Devtools
- Copier-coller (sauf autorisation explicite)
- Fermer la page avant submit

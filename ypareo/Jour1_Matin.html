Contenu de la séance (cours magistral + démonstrations)

Notions abordées :
  • Introduction au module : objectifs, organisation des 4 jours, évaluation
  • L'IA comme fonction mathématique : fθ(x) = ŷ (entrées, sorties, paramètres)
  • Présentation du dataset fil rouge : 30 développeurs, 5 features, prédiction du salaire
  • Vecteurs : représentation d'un développeur comme vecteur x ∈ ℝ⁵
  • Matrices : le dataset comme matrice X ∈ ℝ³⁰ˣ⁵
  • Produit scalaire : xᵀw = somme pondérée, intuition de la « moyenne pondérée »
  • Multiplication matrice-vecteur : X·w = n prédictions simultanées
  • Régression linéaire simple : ŷ = w₁·expérience + w₀, formule des moindres carrés
  • MSE (Mean Squared Error) : mesure de l'erreur du modèle
🎯 Fil rouge — Application sur le dataset salaire
• Chaque développeur du dataset est représenté comme un vecteur de 5 composantes.
• Le dataset complet forme une matrice 30×5.
• On calcule une régression linéaire expérience → salaire : ŷ = 2.07×exp + 31.28.
• MSE = 20.46, soit une erreur typique d'environ 4.5 k€.
Compétences visées :
  • Représenter des données sous forme de vecteurs et matrices
  • Calculer un produit scalaire et interpréter le résultat comme une prédiction
  • Implémenter une régression linéaire simple et calculer le MSE
  • Manipuler des tableaux NumPy (création, slicing, opérations)

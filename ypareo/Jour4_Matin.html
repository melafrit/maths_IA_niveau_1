Contenu de la séance (cours magistral + démonstrations)

Notions abordées :
  • Rappel Jour 3 : le perceptron ne résout pas XOR → il faut combiner plusieurs neurones
  • Le neurone artificiel enrichi : z = wᵀx + b puis a = σ(z) (activation non linéaire)
  • Fonctions d'activation : sigmoïde σ(z) = 1/(1+e⁻ᶻ) (interrupteur progressif), ReLU = max(0,z)
  • Pourquoi les activations sont indispensables : sans elles, tout reste linéaire (démonstration)
  • Composition de fonctions : f∘g(x) = f(g(x)), la sortie d'un neurone alimente le suivant
  • Réseau multicouche (MLP) : couche d'entrée → couche(s) cachée(s) → couche de sortie
  • Forward pass : h = σ(W⁽¹⁾x + b⁽¹⁾), ŷ = σ(w⁽²⁾ᵀh + b⁽²⁾)
  • Exemple numérique complet : réseau 2-2-1 calculé à la main
  • Backpropagation (intuition) : propager l'erreur en arrière, règle de la chaîne appliquée à chaque couche
  • Limites : surapprentissage (overfitting), biais dans les données
  • Synthèse globale des 4 jours : J1 Représentation → J2 Optimisation → J3 Classification → J4 Réseaux
🎯 Fil rouge — Application sur le dataset salaire
• On construit un réseau 2-4-1 from scratch pour classifier salaire_eleve.
• Accuracy du réseau : ~93-97% (vs ~87-90% pour le perceptron du Jour 3).
• Le réseau résout XOR ! Démonstration en live avec surface de décision (contourf).
• La frontière de décision du réseau est courbe (non linéaire), ce qui explique la meilleure accuracy.

  • Saisir l'intuition de la backpropagation (sans calcul complet)

  • Identifier les risques de surapprentissage

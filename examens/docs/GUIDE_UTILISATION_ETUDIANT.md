## 📋 Contenu prévu

Ce guide couvrira :

1. **Accéder à l'examen**
   - Recevoir le code d'examen de votre enseignant
   - Se connecter à la plateforme
   - Saisir vos informations (nom, prénom, email)

2. **Règles à respecter**
   - Rester sur la page de l'examen (focus-lock)
   - Utiliser le mode plein écran
   - Ne pas copier-coller le contenu
   - Conséquences en cas d'infraction

3. **Passer l'examen**
   - Interface de l'examen
   - Navigation entre questions
   - Chronomètre et alertes
   - Soumettre vos réponses

4. **Consulter votre correction**
   - Via le lien dans l'email de confirmation
   - Via upload de votre CSV
   - Comprendre votre note et votre rang
   - Télécharger votre correction en PDF

5. **Vos droits RGPD**
   - Exporter vos données
   - Demander rectification
   - Demander suppression

6. **Support**
   - Que faire en cas de problème technique
   - Contact enseignant

## 📋 Contenu prévu

Ce guide couvrira :

1. **Premier pas**
   - Créer votre compte enseignant
   - Se connecter à la plateforme
   - Tour du dashboard

2. **Gérer la banque de questions**
   - Explorer la banque
   - Créer une question manuellement
   - Importer des questions en masse
   - Utiliser la génération IA
   - Organiser par modules et chapitres

3. **Créer un examen**
   - Wizard en 4 étapes
   - Sélection hybride des questions
   - Configurer les paramètres
   - Partager le code avec les étudiants

4. **Surveiller un examen en cours**
   - Dashboard temps réel
   - Gérer les anomalies
   - Actions (pause, clôture, reset)

5. **Analyser les résultats**
   - Rapport par examen
   - Analyses transversales
   - Exports (CSV, XLSX, PDF, Word)

6. **Administration (rôle admin)**
   - Gérer les comptes enseignants
   - Configurer les paramètres globaux
   - Consulter les logs d'audit

7. **FAQ & Troubleshooting**
   - Questions fréquentes
   - Résolution de problèmes courants
